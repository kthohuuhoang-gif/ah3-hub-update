# AH3 Hub - Updates

Update repository for **AH3 Hub** - All-in-one productivity hub for Autodesk 3ds Max.

## How it works

AH3 Hub checks `version.json` in this repo to detect new versions.  
Update packages (`.ah3` files) are hosted as GitHub Release assets.

## For users

In 3ds Max: **AH3 Hub > Settings > About > Check for Updates**

## version.json

| Field | Description |
|-------|-------------|
| `version` | Latest version string (e.g. `1.8.4.61`) |
| `url` | Direct download URL for the `.ah3` update package |
| `sha256` | SHA-256 hash of the `.ah3` file for integrity check |
| `date` | Release date (YYYY-MM-DD) |
| `notes` | Brief release notes |
